Start with a machine running ubuntu.

start with a fresh install of ubuntu. Make sure you have open-ssh installed and that a hostname has been setup.

Setup no password SSH

use ssh-copy to copy your ssh key over to the new server.

Add scripts to /home/dono/.scripts/

Copy the .scripts folder from the main server over to the new server.

Install VirtualBox

Install virtualbox on the new server using apt-get update; apt-get install virtualbox -y

Import VM

use scp to copy the ubuntu folder from the main server to your new server.
